src¶

src module contains the following source code files:

src.main¶

This code contains the numerical experiments with the step adaptive RKMK method based on Dormand-Prince pair (5,4)

src/integrators¶

src.integrators.RKMK5(vecField, exponentialSE3N, action, p, h, sigma0)¶

Runge-Kutta-Munthe-Kaas time integrator order 5

Parameters:

  • vecField – right hand side of the ODE

  • exponentialSE3N – exponential map from se(3)^N to SE(3)^N

  • action – Lie group action

  • p – solution at time t_n

  • h – time step size

  • sigma0 – initial value of the curve sigma on se(3)^N

Returns:

solution at time t_(n+1)

src.integrators.RKMK45(vecField, exponentialSE3N, action, p, h, sigma0)¶

Runge-Kutta-Munthe-Kaas pair (5,4) based on the Dormand-Prince method DOPRI5(4)

Parameters:

  • vecField – right hand side of the ODE

  • exponentialSE3N – exponential map from se(3)^N to SE(3)^N

  • action – Lie group action

  • p – solution at time t_n

  • h – time step size

  • sigma0 – initial value of the curve sigma on se(3)^N

Returns:

solution at time t_(n+1) and local error
